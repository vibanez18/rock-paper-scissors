# ROCK, PAPER, SCISSORS

Small project based on the classic game rock, paper and scissors. The tech stack is:

- Angular (frontend)
- Kotlin 1.7
- Spring Boot 3

## How to run
### Frontend
Execute the command in rock-paper-scissor-ui folder.

    ng serve --open

### Backend
Execute the command in the root folder.

     ./gradlew bootRun
